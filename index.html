Project 25%: Application of Derivatives in Bioscience

SFA | SFH | SFM | SFP Program Requirements

Project Overview

Create an HTML animation showing the Application of Derivatives in Bioscience in Malaysia related to your program. You will upload this to GitHub as a public webpage, present via Google Meet, and publish the recording to YouTube.

Task 1: Develop

Reference Example:

View a sample project of how derivatives are applied to fertilizer efficiency:

View Sample Webpage Application

Use gemini.google.com (Tools: Canvas) to generate your index.html. Copy and paste the following prompt:

PROMPT FOR GEMINI "I am an undergraduate student in Malaysia majoring in [Insert Program: SFA: Agrotech, SFH: Animal Husbandry Technology, SFM: Food Security, SFP: Agro Product Technology]; please act as a web developer and bioscience expert to create a single-file 'index.html' application that uses a Chart.js animation to demonstrate the application of derivatives f'(x) in a realistic Malaysian bioscience scenario relevant to my field (such as crop growth, animal weight gain, or food processing rates). The application must include the function f(x) and derivative f'(x) formulas, a clear explanation of the bioscience problem, use real or realistic data points with the data webpage names, and allow the user to click on the graph to display the calculated rate of change (derivative) and layman short explaination at that specific point."

Task 2: Deploy

1. Copy the generated HTML code.

2. Paste into Notepad and save as index.html.

3. Upload to GitHub and enable GitHub Pages. Use this prompt for guidance:

"I am a total beginner with GitHub. I have finished creating an index.html file on my computer desktop for my research project. I need you to act as my technical tutor. Please give me a step-by-step guide, use numbered instructions so I don't get lost, on how to host this file as a live, public website using GitHub Pages. Tell me how to find the final website link so I can copy and give to my friend."

Task 3: Present

  • LEADER: Decide a presentation date and time.
  • LEADER: Create and arrange Google Meet session.
  • EDITOR: Record the whole session.
  • EVERYONE: Present your webpage from your live GitHub URL (4-6 mins each).

Presentation Flow:

  1. Self Introduce
  2. Summary of the chosen bioscience problem
  3. Importance of studying this bioscience problem
  4. Click and explain meaning of each point
  5. Recommend one way to make this HTML better

Task 4: Publish

After completing Google Meet, EDITOR to edit the video (basic edit, no fancy editing). EDITOR to get YouTube Description from DESCRIBER, then upload to YouTube (Public).

Marking Criteria Details

Problem Selection and Explanation (5%)

5 = Perfect alignment with program (SFA/SFH/SFM/SFP); Malaysian context is clear; explanation of derivative relevance is insightful.

Derivative Solution (5%)

5 = HTML code correctly calculates the derivative; uses accurate bioscience data; logic is flawless.

Communication and Presentation (10%)

10 = Professional Google Meet delivery; YouTube video is clear with a working GitHub link in the description.

Overall Quality and Creativity (5%)

5 = High-quality HTML animation; seamless GitHub hosting; highly innovative bioscience scenario.
